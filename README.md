# Online Voting System: An Online Voting System built using **Spring Boot**, **Java**, and **MySQL** that allows users to securely vote online and enables administrators to manage elections, candidates, and results.
This project demonstrates backend development concepts such as REST APIs, authentication, database integration, and role-based access.
## Tech Stack
| Layer        | Technology |
|-------------|------------|
| Backend     | Java, Spring Boot |
| Database    | MySQL |
| ORM         | Spring Data JPA |
| API Testing | Postman |
| Build Tool  | Maven |
| Server      | Embedded Tomcat |
